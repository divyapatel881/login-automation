# Login Automation Test Suite

Automated UI tests for a login page using Python and Selenium WebDriver.

## Tests included
- Smoke test: verifies login page loads correctly
- Regression: valid login flow
- Regression: invalid username error
- Regression: invalid password error
- Regression: empty fields validation

## Tech stack
- Python 3.x
- Selenium WebDriver
- WebDriver Manager
- Chrome browser

## How to run
pip install selenium webdriver-manager
python test_login.py
